What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents, specifically HTML and XML. It represents the structure of a document as a tree of objects.

How do you select an element from the DOM?

Selecting an element from the DOM can be done in a few different ways using JavaScript. Here are some common methods:
1.getElementById()
2.getElementsByClassName()
3.getElementsByTagName()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a powerful and efficient technique in DOM manipulation. By handling events at a higher level in the DOM , it reduces the need for multiple event listeners, handles dynamic elements seamlessly, and improves overall performance.

How do you manipulate an element's attributes and styles using the DOM?

Manipulating an element's attributes and styles using the DOM is a common task in web development. JavaScript provides several methods to change both attributes (like id, src, href, etc.) and styles (CSS properties) dynamically.
